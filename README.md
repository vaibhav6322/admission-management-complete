
#  (Complete Enterprise MERN)

## Features
- JWT Authentication + Role-based access
- Master Program + Quota validation
- Applicant Management
- Atomic Seat Allocation
- Admission Confirmation (Fee required)
- Real-time Dashboard stats
- Redux Toolkit integration
- Axios interceptor
- Postman collection
- Dockerized setup

## Run with Docker
docker-compose up --build

## Manual Run

Backend:
cd server
npm install
npm run dev

Frontend:
cd client
npm install
npm run dev
